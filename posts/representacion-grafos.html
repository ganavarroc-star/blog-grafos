Representación de Grafos

Las dos formas principales de representar grafos son:

1. Lista de Adyacencia

Eficiente para grafos dispersos (pocas conexiones).

A: B, D
B: A, C
C: B, D, E
D: A, C
E: C

2. Matriz de Adyacencia

Eficiente para consultar si una arista existe.

    A B C D E
A [ 0 1 0 1 0 ]
B [ 1 0 1 0 0 ]
C [ 0 1 0 1 1 ]
D [ 1 0 1 0 0 ]
E [ 0 0 1 0 0 ]
